This is a demo puzzle that ships with the site so you can see how everything
works. It's based on the [Sator Square](https://en.wikipedia.org/wiki/Sator_Square),
a Latin palindromic word square found scratched into walls across the Roman
Empire — the same five words read across and down.

Delete this folder once you've added your first real puzzle.
